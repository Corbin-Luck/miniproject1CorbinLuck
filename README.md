# Project Title
 
Mini Project 1 Stock Tickers
 
## Description
 
The program looks up 5 stock tickers and generates 5 graphs of the closing price for the last 10 business days for the stock tickers.
 
## Getting Started
 
### Dependencies

 Please install the pip requirements:
```
pip install -r requirements.txt
```
 
### Executing program
 
```
python main.py
```
